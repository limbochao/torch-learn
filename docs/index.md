# torch-learn

`torch-learn` 是 PyTorch 学习、需求设计、测试、问题定位和技能沉淀的统一入口。

## 内容入口

- [学习笔记](notes/)
- [需求设计](requirements-design/)
- [问题定位日志](issue-logs/)
- [脚本索引](scripts/)
- [Torch skills](skills/)

## 仓库目标

1. 记录 PyTorch 学习过程中的关键概念、源码分析和实验结论。
2. 沉淀需求背景、方案设计、接口约束、验证计划和风险记录。
3. 保存可复现的测试脚本、最小复现脚本和调试辅助脚本。
4. 沉淀问题定位日志，保留现象、分析路径、结论和验证结果。
5. 维护 Torch 相关 skills，方便 Codex 在后续分析和开发中复用。
6. 逐步承接并替代 `torch_scripts` 中仍有价值的内容。
